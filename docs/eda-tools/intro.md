# EDA工具概述

## 什么是EDA工具？

EDA（Electronic Design Automation）工具是用于电子设计自动化的软件工具，是芯片设计不可或缺的辅助工具。

## 主要EDA厂商

### 1. Synopsys
- Design Compiler：综合工具
- PrimeTime：静态时序分析
- VCS：仿真工具
- SpyGlass：代码检查

### 2. Cadence Design Systems
- Virtuoso：模拟/混合信号设计
- Allegro：PCB设计
- Genus：综合工具
- Incisive：仿真平台

### 3. Siemens EDA（原Mentor Graphics）
- Calibre：物理验证
- ModelSim：仿真工具
- Precision：综合工具

### 4. 开源工具
- OpenROAD：数字设计流程
- KLayout：GDSII查看器
- Ngspice：电路仿真

## 工具分类

### 前端设计工具
- HDL编辑器
- 仿真工具
- 代码检查工具
- 综合工具

### 后端设计工具
- 布局布线
- 时序分析
- 物理验证（DRC/LVS）
- 寄生参数提取

:::tip 工具选择
根据项目需求和预算选择合适的EDA工具组合。开源工具在教育和研究中越来越受欢迎。
:::
