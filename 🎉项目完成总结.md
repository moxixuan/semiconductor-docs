### 步骤1：下载项目

```bash
# 位置：/home/moxixuan/semiconductor-docs/
# 下载文件：~/semiconductor-docs.tar.gz (29MB)

# 下载命令：
scp user@your-server:~/semiconductor-docs.tar.gz ~/
```

### 步骤2：解压并安装

```bash
# 解压
cd ~
tar -xzf semiconductor-docs.tar.gz
cd semiconductor-docs

# 检查Node版本（需要>=20.0）
node --version

# 安装依赖
npm install --legacy-peer-deps
```

### 步骤3：启动查看效果

```bash
# 方法A：自动化脚本（推荐）
./SETUP_GUIDE.sh

# 方法B：手动启动
npm start

# 访问 http://localhost:3000
```
