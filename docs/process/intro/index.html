工艺与制造概述

半导体制造工艺简介​

半导体制造工艺是将设计好的电路图案转移到硅晶片上的过程。

主要工艺环节​

1. 晶圆制备​

  • 提拉单晶硅
  • 晶圆切割
  • 抛光清洗

2. 氧化与沉积​

  • 热氧化
  • 化学气相沉积（CVD）
  • 物理气相沉积（PVD）

3. 光刻​

  • 涂胶
  • 曝光
  • 显影
  • 刻蚀

4. 掺杂​

  • 离子注入
  • 扩散

5. 金属化​

  • 金属沉积
  • 导通孔形成
  • 化学机械抛光（CMP）

工艺技术节点​

先进节点​

  • 7nm/5nm：当前主流先进工艺
  • 3nm：最新工艺节点
  • FinFET：鳍式场效应晶体管

成熟节点​

  • 28nm
  • 40nm
  • 65nm
  • 90nm
技术趋势

工艺节点越来越小，设计规则越来越复杂，对EDA工具的要求也越来越高。
